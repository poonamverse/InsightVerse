# InsightVerse – A Multi-Theme Blog Platform

##  Overview
InsightVerse is a dynamic blog web application built using Node.js, Express.js, and EJS. It allows users to create, view, edit, and delete blog posts across multiple themes including:

🎬 Movies (e.g., 3 Idiots)
🏏 Sports & Motivation (inspired by Virat Kohli)
🧠 Psychology (real-life insights and thoughts)

The project focuses on combining meaningful content with a clean, responsive user interface.

##  Features
- Create, edit, and delete blog posts
- View all posts on homepage
- Category-based content (Movies, Sports, Psychology)
- Responsive UI design

## Tech Stack
- Node.js
- Express.js
- EJS
- CSS

## Setup
```bash
npm install
node app.js
```

## 📌 Note
This project stores data in-memory for simplicity and resets on server restart. Future versions may include database integration.
